Here's my Infographic Project

The infographic project was a very interesting project to work on. This project required choosing a topic and creating an informational poster with visually pleasing features and imagery. Our topics were supposed to relate to a communication system, so I decided to do mine on how the guitar amplification system works. This model shows an example of machine to machine communication. I started with the origins of how it came to be and moved to how each of the systems worked and communicated with eachother. I then finished with impacts and innovations.
